First-time setup (~2 minutes):
  1. Create an account at fal.ai and add a payment method under Billing (pay-per-use, no subscription).
  2. Go to Dashboard → Keys, create a key, and paste it in the field above.
  3. The key is stored only in this browser (localStorage) — nothing is sent anywhere except directly to fal.ai.
